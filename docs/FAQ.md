# Secure File Share - Frequently Asked Questions

## General Questions

### What is Secure File Share?

Secure File Share is a simple, secure platform for uploading and sharing files with anyone using unique numeric codes. No account required to get started, and files are automatically deleted after expiration for privacy.

### How does it work?

1. Upload a file
2. Receive a unique 9-digit share code
3. Share the code with anyone
4. Recipients enter the code to download
5. File automatically deletes after expiration

### Is it free?

Yes! We offer a free plan with basic features. We also offer paid plans with more storage and longer expiration times.

### What file types are allowed?

**Allowed:**
- Documents: PDF, DOCX, XLSX, PPTX, TXT, RTF, ODT
- Images: PNG, JPG, JPEG, GIF, WEBP, SVG, BMP
- Archives: ZIP, RAR, 7Z, TAR, GZ
- Media: MP4 (50MB max), WEBM (50MB max), MP3, WAV, OGG, M4A
- Code: JSON, XML, CSV, SQL (in archives only)

**Not Allowed:**
- Executables: EXE, BAT, SH, COM, DLL, MSI, APP, DMG
- Scripts: VBS, PS1, PY (standalone), RB (standalone)
- System files: SYS, DRV, INF, REG, SCR

### What's the maximum file size?

- Free plan: 100MB
- Paid plan: 1GB
- Enterprise plan: 10GB

### How long are files kept?

- Free plan: 20 minutes
- Paid plan: 24 hours
- Enterprise plan: Custom (up to 30 days)

## Account & Authentication

### Do I need an account to upload?

No! You can upload files without creating an account. However, creating an account lets you track your files and access additional features.

### How do I create an account?

1. Click "Sign Up" button
2. Enter your email address
3. Create a strong password
4. Verify your email
5. Account is ready to use

### What are the password requirements?

- At least 8 characters
- Include uppercase letter (A-Z)
- Include lowercase letter (a-z)
- Include number (0-9)
- Include special character (!@#$%^&*)

Example: `SecurePass123!`

### How do I reset my password?

1. Go to login page
2. Click "Forgot Password"
3. Enter your email
4. Check email for reset link
5. Click link and create new password

### Can I change my email address?

Yes! Go to Account Settings and click "Change Email". You'll need to verify both your old and new email addresses.

### How do I delete my account?

1. Go to Account Settings
2. Scroll to bottom
3. Click "Delete Account"
4. Confirm deletion
5. All data is permanently deleted

## Uploading Files

### How do I upload a file?

1. Go to upload page
2. Click upload area or drag file
3. Complete CAPTCHA verification
4. Click "Upload"
5. Wait for upload to complete
6. Receive share code

### Why does upload fail?

Common reasons:
- File too large (exceeds plan limit)
- File type not allowed
- Internet connection dropped
- Rate limit exceeded (free plan: 5/day)
- Virus detected in file

See Troubleshooting Guide for solutions.

### Can I upload multiple files at once?

Not directly, but you can:
- ZIP multiple files together
- Upload ZIP archive
- Recipients extract after download

### Can I upload from mobile?

Yes! Works on all modern mobile browsers. You can also install the mobile app for better experience.

### What happens if upload is interrupted?

Upload will fail and you'll need to try again. We recommend:
- Using wired connection if possible
- Uploading during off-peak hours
- Compressing large files first

### Can I edit a file after uploading?

No, files cannot be edited after upload. You can:
- Delete the file
- Upload a new version with different share code

## Downloading Files

### How do I download a file?

1. Go to download page
2. Enter 9-digit share code
3. Click "Find File"
4. Review file information
5. Click "Download"
6. File downloads to your device

### Can I download a file multiple times?

Yes! Files can be downloaded unlimited times until expiration.

### What if the file has expired?

Files automatically delete after expiration. You'll need to ask the sender to upload again.

### Can I download without the share code?

If you have the direct link, you can download directly without entering the code.

### Why can't I download?

Common reasons:
- Share code is incorrect
- File has expired
- File was deleted by uploader
- Internet connection issue

### Can I download on mobile?

Yes! Works on all modern mobile browsers and the mobile app.

## Sharing Files

### How do I share a file?

After uploading, you can:
- Copy share code and send via email/chat
- Copy direct link and share
- Share QR code
- Share via social media

### Is my share code private?

Yes! Only people with the share code can access the file. Share codes are not publicly listed.

### Can I share with specific people?

Not directly, but you can:
- Share code only with intended recipients
- Use direct link for trusted recipients
- Share QR code in person

### Can I track who downloaded my file?

Yes! If you have an account, go to Dashboard to see:
- Download count
- Download dates
- Downloader locations (approximate)
- Downloader devices

### Can I prevent downloads?

No, but you can:
- Delete file manually
- Wait for automatic expiration
- Use shorter expiration time

## Security & Privacy

### Is my file secure?

Yes! We use:
- HTTPS/TLS encryption in transit
- AES-256 encryption at rest
- Virus scanning on all uploads
- Automatic deletion after expiration

### Who can access my files?

Only people with the share code can access your files. Files are not publicly listed or searchable.

### Do you scan files for viruses?

Yes! All files are scanned for malware before storage. Infected files are rejected and not stored.

### What happens to my data?

- Files are automatically deleted after expiration
- Metadata is deleted with file
- We don't sell your data
- See Privacy Policy for details

### Is my data GDPR compliant?

Yes! We comply with GDPR and offer:
- Data export on request
- Data deletion on request
- Data Processing Agreement
- Privacy policy

### Can I encrypt files myself?

Yes! You can:
- Encrypt file before uploading
- Share encryption key separately
- Recipients decrypt after download

### What about 2FA?

Two-factor authentication coming soon! We'll support:
- TOTP (Google Authenticator, Authy)
- SMS codes
- Email codes
