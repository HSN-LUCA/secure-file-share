## Getting Started

### What You Need

- A web browser (Chrome, Firefox, Safari, Edge)
- A file to upload (up to 100MB for free users)
- Internet connection

### First Time?

1. Visit **secure-file-share.com**
2. You'll see the upload interface
3. No login required to start sharing!

## Uploading Files

### Step-by-Step Upload

#### 1. Select Your File

**Option A: Click to Browse**
- Click the upload area
- Select a file from your computer
- File preview will appear

**Option B: Drag and Drop**
- Drag a file from your desktop
- Drop it onto the upload area
- File preview will appear

#### 2. Verify File Details

Before uploading, you'll see:
- File name
- File size
- File type
- Estimated expiration time

#### 3. Complete Security Check

- A CAPTCHA verification will appear
- Click the checkbox to verify you're human
- This protects against automated abuse

#### 4. Upload Your File

- Click the **Upload** button
- Progress bar shows upload status
- Wait for upload to complete

#### 5. Get Your Share Code

After successful upload, you'll receive:
- **Share Code**: A unique 9-digit number
- **Download Link**: Direct link to share
- **Expiration Time**: When file will be deleted

## Sharing Files

### Share QR Code

1. Click **QR Code** button
2. Scan with phone camera
3. Share QR code image

## Downloading Files

### Using Share Code

#### 1. Go to Download Page

- Visit **secure-file-share.com/download**
- Or click a shared link

#### 2. Enter Share Code

- Enter the 9-digit share code
- Code is case-insensitive
- Click **Find File**

#### 3. Review File Info

You'll see:
- File name
- File size
- Upload date
- Expiration time

#### 4. Download File

- Click **Download** button
- File will download to your device
- Download counter increments

### Using Direct Link

- Click the shared link
- File info appears automatically
- Click **Download** to start

### Mobile Download

- Tap the download link
- File opens in browser or downloads
- Long-press to save to device

## Creating an Account

### Why Create an Account?

**Free Account Benefits:**
- Track your uploaded files
- View download statistics
- Extend file expiration
- Delete files manually
- Access file history

**Paid Plan Benefits:**
- 1GB file size limit (vs 100MB)
- 24-hour expiration (vs 20 minutes)
- Unlimited uploads (vs 5/day)
- Advanced analytics
- Priority support

### Registration Steps

#### 1. Click Sign Up

- Click **Sign Up** button in top right
- Or go to **secure-file-share.com/signup**

#### 2. Enter Email

- Enter your email address
- Must be valid email format
- You'll receive confirmation email

#### 3. Create Password

Password requirements:
- At least 8 characters
- Include uppercase letter (A-Z)
- Include lowercase letter (a-z)
- Include number (0-9)
- Include special character (!@#$%^&*)

Example: `SecurePass123!`

#### 4. Verify Email

- Check your email inbox
- Click verification link
- Account is now active

#### 5. Log In

- Enter email and password
- Click **Log In**
- You're now authenticated

## Understanding Plans

### Free Plan

**Cost:** Free forever

**Limits:**
- File size: 100MB
- Expiration: 20 minutes
- Uploads: 5 per day
- Storage: 500MB total

**Features:**
- Basic upload/download
- Share codes
- File history (with account)

### Paid Plan

**Cost:** $4.99/month or $49.99/year

**Limits:**
- File size: 1GB
- Expiration: 24 hours
- Uploads: Unlimited
- Storage: 100GB total

**Features:**
- Everything in Free plan
- Extended expiration
- Advanced analytics
- Priority support
- API access

### Upgrading to Paid

1. Log in to account
2. Click **Upgrade** button
3. Select plan duration
4. Enter payment information
5. Subscription activates immediately

### Downgrading

1. Go to **Account Settings**
2. Click **Manage Subscription**
3. Click **Cancel Subscription**
4. Downgrade to Free plan
5. Existing files remain until expiration

## Security & Privacy

### How We Protect Your Files

**Encryption in Transit:**
- All uploads use HTTPS/TLS
- Data encrypted during transfer
- Cannot be intercepted

**Encryption at Rest:**
- Files encrypted in storage
- AES-256 encryption
- Only you can access your files

**Virus Scanning:**
- All files scanned for malware
- Infected files rejected
- Your safety is our priority

**Automatic Deletion:**
- Files auto-delete after expiration
- No manual cleanup needed
- Protects your privacy

### Privacy Policy

- We don't sell your data
- No tracking cookies
- Minimal analytics
- GDPR compliant
- See full policy: secure-file-share.com/privacy

### What We Don't Store

- Your file contents (after deletion)
- Payment information (Stripe handles it)
- Browsing history
- Personal information beyond email

## Tips & Tricks

### Best Practices

**For Sharing:**
- Use direct links for trusted recipients
- Share codes via secure channels
- Set reminders for file expiration
- Delete sensitive files manually

**For Security:**
- Use strong passwords
- Don't share links publicly
- Log out on shared devices
- Enable 2FA (coming soon)

**For Performance:**
- Compress large files before upload
- Use ZIP for multiple files
- Upload during off-peak hours
- Check file size before uploading

### Troubleshooting

**Upload Fails**
- Check file size limit
- Verify file type allowed
- Check internet connection
- Try different browser

**Can't Download**
- Verify share code is correct
- Check if file expired
- Try direct link instead
- Clear browser cache

**Slow Upload**
- Check internet speed
- Close other applications
- Try wired connection
- Upload smaller file

**Account Issues**
- Check email for verification link
- Reset password if forgotten
- Clear browser cookies
- Try incognito mode

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+U | Go to upload page |
| Ctrl+D | Go to download page |
| Ctrl+L | Copy share link |
| Ctrl+C | Copy share code |

## Frequently Asked Questions

**Q: How long are files kept?**
A: Free plan: 20 minutes. Paid plan: 24 hours. Enterprise: Custom.

**Q: Can I download a file multiple times?**
A: Yes! Files can be downloaded unlimited times until expiration.

**Q: What file types are allowed?**
A: PDF, images, documents, archives, and media files. See full list in API docs.

**Q: Is my file private?**
A: Yes! Only people with the share code can access it.

**Q: Can I change the expiration time?**
A: Paid plan users can extend expiration. Free plan cannot.

**Q: What happens if I don't download before expiration?**
A: File is automatically deleted. No recovery possible.

**Q: Can I upload from mobile?**
A: Yes! Works on all modern mobile browsers and the mobile app.

**Q: Is there a file size limit?**
A: Free: 100MB. Paid: 1GB. Enterprise: 10GB.
